# Casino Bonus Comparison Website

A comprehensive casino bonus comparison platform with real-time aggregation, filtering, and affiliate tracking.

## Features
- Real-time bonus aggregator from major casinos
- Country-specific filtering (AU, CA, IE, UK, etc.)
- Bonus type filtering (no deposit, welcome, free spins)
- Wagering requirement calculator
- User reviews and ratings system
- Affiliate tracking integration
- Responsive design with casino theme
- SEO optimized with high-value keywords
- Schema markup implementation
- Fast loading and conversion-optimized CTAs

├── scripts/           # Data aggregation and maintenance scripts

## Target Domain: casino-bonuses-guide.com

## Technology Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Node.js with Express
- Database: SQLite for development, PostgreSQL for production
- Real-time updates: WebSocket
- SEO: Schema.org markup, meta optimization
- Analytics: Google Analytics 4, affiliate tracking
